## 프로젝트 개요

- 이 프로젝트는 **Nomadcoder**의 [코코아 클론코딩](https://nomadcoders.co/kokoa-clone) 강의 실습을 바탕으로 제작 되었습니다.

   Frontend 부분을 구현합니다.
